2 Cara Mendapatkan QR Code di Swagger

Setelah Swagger UI terbuka:

  1. Cari section "Sessions" atau "Auth"
  2. Cari endpoint yang berkaitan dengan QR atau screenshot
  3. Klik endpoint tersebut untuk expand
  4. Klik tombol "Try it out"
  5. Masukkan session name: default
  6. Klik "Execute"
  7. QR code akan muncul di response (bisa sebagai image atau base64)
