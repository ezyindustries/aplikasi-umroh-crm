Network Access

To access this dashboard from other devices on your network:

  1. Find your computer's IP address (e.g., 192.168.1.100)
  2. Make sure the backend server is running on port 5000
  3. Access from other devices using: http://YOUR_IP:5000/frontend/crm-no-login.html
  4. Ensure firewall allows connections on port 5000
